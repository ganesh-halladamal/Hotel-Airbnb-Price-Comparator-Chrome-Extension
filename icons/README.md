# Extension Icons

This folder contains the icons for the Chrome extension. You'll need to add the following icon files:

## Required Icons
- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Design Guidelines
- Use a hotel/accommodation theme
- Include price comparison elements (💰, 📊, 🔍)
- Ensure good visibility on different backgrounds
- Follow Chrome extension icon best practices

## Temporary Icons
You can use any hotel/travel-related icons temporarily:
- 🏨 Hotel building
- 💰 Price/money symbols
- 🔍 Magnifying glass
- 📊 Chart/graph

## Creating Icons
1. Use tools like Figma, Canva, or Photoshop
2. Export as PNG with transparent background
3. Ensure crisp rendering at all sizes
4. Test on both light and dark browser themes
